# Workflow Package

This package builds boilerplate commands for the `ok workflow` CLI. The templates it references live in the golden-path-boilerplate repo.

## Related Templates

When modifying this package, check the corresponding boilerplate templates for consistency:

- `boilerplate/github-actions/app-cicd` — app CI/CD workflow template
- `boilerplate/github-actions/terraform-iac` — IAC workflow template

Ask the user if they have a local clone of golden-path-boilerplate. If so, use that. Otherwise, reference https://github.com/oslokommune/golden-path-boilerplate.
